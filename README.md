## Getting Started
To get a local copy of the repository please run the following commands on your terminal:
- ```$ cd <folder>```
- ```$ git clone https://github.com/medobarakat/Film-Shows-Page.git ```
- ```$ cd Film-Shows-Page ```
- ```npm install```
- ```npm start```
